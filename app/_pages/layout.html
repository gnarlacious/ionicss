Container

The container is the frame that dictates the max-width the entire website.

Usage

{% highlight scss %} @include container(); {% endhighlight %}

Options

Change how the container is used.

{% highlight scss %} @include container(( name: 'wrapper', max-width: $bp-largest )); {% endhighlight %}

Grid

Grid Variables

The grid has several default styles and naming structure. But all areas of the grid are capable of changing.

Namespacing

Rename the grid if the project needs to with namespaces. The default values given are row and column.

{% highlight scss %} $grid-namespace: 'row'; $grid-column-namespace: 'column'; {% endhighlight %}

Column Groups

The default columns for a 12 grid framework.

{% highlight scss %} $col-groups: (2, 3, 4, 5, 6, 8, 12); {% endhighlight %}

Gutter Width

Gutter width is used both horizontally and vertically bottom spacing.

{% highlight scss %} $gutter-width: 20px; {% endhighlight %}

Pushes and Pulls

A boolean for column ordering. By default is set to true. If column ordering is not necessary set these variables to false to save on file space.

{% highlight scss %} $pushes: true; $pulls: true; {% endhighlight %}

Column Ordering

For mobile-friendly ordering add --push and --pull modifiers to the column class.

.column--2-of-5.column--push-3-of-5
.column--3-of-5.column--pull-2-of-5
{% highlight html %}
.column--2-of-5.column--push-3-of-5
.column--3-of-5.column--pull-2-of-5
{% endhighlight %}
