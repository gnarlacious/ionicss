Variables

Override default variables above the import of {{site.name}}.scss file.

Font Size

Choose whether to use the default variables given or change them to reflect your design.

{% highlight scss %} $font-size-base: 14px; $line-height-base: 1.4; // Element Specific $font-size-blockquote: 18px; {% endhighlight %}

Font Families

Base Styles

Change base font families to keep it simple.

{% highlight scss %} $font-family-primary: sans-serif; $font-family-secondary: serif; $font-family-code: monospace; {% endhighlight %}

Specific Styles

To get more specific with font families, change every section separately.

{% highlight scss %} $font-family-base: $font-family-primary; $font-family-action: $font-family-primary; $font-family-heading: $font-family-secondary; $font-family-lead: $font-family-secondary; $font-family-blockquote: $font-family-secondary; {% endhighlight %}

Font Weights

Each section has a specific weight attached to it as a default. You may override those as well.

{% highlight scss %} $font-weight-base: 400; $font-weight-action: 400; $font-weight-heading: 700; $font-weight-lead: 400; {% endhighlight %}

Quotes

There are three different types of quotes. There is the original blockquote everyone knows and loves with the addition of pullquotes to either side of the document.

Blockquote

The basic blockquote with minimal tweaks.

{% highlight html %}

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis.
{% endhighlight %}

Pullquote Directional

Pullquotes are essentailly blockquotes with a modifier for directional purposes.

{% highlight html %}

Lorem ipsum dolor sit amet, consectetur adipisicing
{% endhighlight %}

Example

Changed three variables above ionicss stylesheet to override the overall typography.

{% highlight scss %} $font-family-primary: "Merriweather"; $font-family-secondary: "Merriweather"; $font-family-action: "Source Sans Pro"; @import 'path/to/ionicss'; {% endhighlight %}
