Installation

{% highlight bash %} $ bower install {{ site.name }} {% endhighlight %}

File path will primarily be located in _bower-components/ionicss/scss/ionicss.scss.

Setup

Import the iconicss.scss into your main sass style sheet.

{% highlight scss %} // styles.scss @import "/path/to/ionicss"; {% endhighlight %}

Includes

Global styles such as normalize, typography, and grid automatically compile. To keep the styles as lightweight as possible no modules or layout are compiled, by design. Every module can be individually included below the ionicss.scss or all of them can be included to use the full capacity of {{ site.name | capitalize }}.

Individual Modules

Only use what you need. Something as simple as a login page with a form, submit button would like like this.

{% highlight scss %} // styles.scss @import "/path/to/ionicss.scss"; @include forms(); @include buttons(); {% endhighlight %}

All Modules

{% highlight scss %} @import "/path/to/ionicss.scss"; @include all(); {% endhighlight %}

Variables

Each element in Ionicss has a set of variables that are capable of overriding. From global typography to layout to every module there will be a list of variables you are capable of overriding.

{% highlight scss %} // styles.scss $font-family-primary: "Merriweather" serif; $input-bkg: #ccc; @import "/path/to/ionicss.scss"; {% endhighlight %}
