Creates basic styles all form elements.

Usage

{% highlight scss %} @include forms(); {% endhighlight %}

Options

{% highlight scss %} @include forms(( name: 'form', label: ( weight: 700, size: $font-size-base, color: $form-label-color ), input: ( border: $input-border, background: $input-bkg, color: $input-color ), help-text-color: #aaa )); {% endhighlight %}

Example

Form inputs have a width of 100% with the purpose of sizing within the grid system.
